# FPV-DRONE-MANAFLY
3" drone project based on 3d printed chasis by manafly. Modern drone chassis are usually always made from carbon fiber, it is light and durable. Using ai generative design, this drone frame is optimized for weight and strength. It's design allows it to perform competitively with carbon fiber frames while being printed from PETG or ABS filament. Each frame costs around $1 in filament. This is the foundation for my project, all of the parts were carefully picked, I spent hours making sure each part worked in tandem with others. I did my best to find the cheapest option for each component, The controller is the cheapest in the market and the goggles are second hand. I wanted to build this project because I was looking for a challenge. My friend and I are both going into engineering and we wanted an ambitious project to have under our belt. I wanted to do this for the experience and the challenge. Seeing this generative design drone chassis inspired me to starting this, it caught my interest and is a very unique concept. I anxiously wait to test it out.
